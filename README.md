# Todo List

A simple [todo list](https://thawing-refuge-22442.herokuapp.com/) app for adding, deleting and editing tasks.

### Install   
```sh
$ yarn install
```   
   
### Run   
```sh
$ yarn start   
```   

### Libraries used  

* React   
* TypeScript   
* MobX     
* Styled Components   
* i18next
